# 🧩 项目决策

> 每个关键选择**为什么这么定**——答辩官最爱追问的就是这些。每条决策写清：选项 / 取舍依据 / 代价 / 如果重来。完整决策记录见 [[执行计划]]。

## 已定案的关键决策

### 执行器：STS3215 ×11

- **为什么**：¥100/个、LeRobot 同款生态成熟、`orca_core` 原生兼容、自带磁编 + 负载反馈，单舵机集成位置环
- **代价**：力矩 1.9 N·m 偏小、外形与官方塔不配 → 需重设计基座
- **待写**：和 Dynamixel 的对比；为什么不用无刷 + 外置驱动

### 五指 → 三指裁剪

- **为什么**：成本 / 装配 / 训练复杂度全降；三指足以覆盖捏取与基础手内操作
- **风险**：三指能否完成手内操作（in-hand manipulation）——**仿真先行验证**后再批量采购
- **待写**：三指可行性的文献/仿真依据；和"对置拇指最小手"的关系

### 传动：平板基座 + 绞盘腱鞘

- **为什么**：STS3215 外形塞不进官方塔；绞盘 + 腱让电机位置自由布局
- **代价**：引入腱的弹性/迟滞，标定更复杂
- **待写**：腱传动的回差/张力管理
