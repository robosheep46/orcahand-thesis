## 0. 课题定案

**题目（暂定）**：《低成本三指灵巧手平台的构建与基于学习的操作技能研究》

**一句话**：把 ETH ORCA 开源手（现行版官方即用飞特 STS3215）裁剪成三指，硬件线与仿真线双轨并行，跑通 遥操作 → 模仿学习（保底）→ RL sim2real（主线）的算法阶梯；触觉与 FOC 执行器升级为可选冲高项。

**定位**：嵌入式岗 + 算法加分，受众机器人公司。

**论文贡献点**：
1. 三指裁剪构型的可行性验证（仿真先行：批量采购前用 RL 验证三指能否完成手内操作）
2. 力矩 vs 位置动作空间的 sim2real 对照实验（仿真内完成，ORCA 论文未做）
3. 低成本（≈¥2400）三指平台上 IL + RL sim2real 全链路复现与 gap 分析
4. （可选）拇指 2 轴 FOC 化的执行器对比研究

## 2. ORCA 开源资产（github.com/orcahand，MIT）

- **orca_core**：Python 控制栈，原生 Dynamixel + Feetech 双后端，YAML 配置，张紧/标定/零位脚本
- **orcahand_description**：MJCF/URDF（三指裁剪 = 改 XML，非 CAD）
- **orca_sim**：仿真环境；论文 RL 基线 = IsaacGymEnvs + A2C + 域随机化，1 小时训出五指转球零样本上实物
- **orca_retargeter / orca_teleop**：人手→机械手重定向（输入端换 MediaPipe，不买手套）
- **orca_ui**：触觉可视化（冲高期用）
- CAD/STL/BOM/装配视频/Bambu 打印参数：orcahand.com/files（需注册下载，**唯一未拿到的资料，待用户注册下载**）；docs.orcahand.com 装配页尚为 Coming soon
- 原版规格备查（源自本地 orca_core 文档/配置，已实锤）：17 DoF；**v2 = Feetech STS3032 ×17**（6-12.6V，堵转 3.5kg·cm，半双工 TTL 1Mbps，可回读位置/速度/电流/温度）；v1 = Dynamixel XL330/XC330 系；控制模式 `current_based_position`（带电流反馈，张力实验有官方支持）；官方标定为"逐关节 flex/extend 自动找行程"序列
- 三指裁剪实锤（v2 config.yaml）：拇指 cmc/abd/mcp/dip ×4 + 食指 abd/mcp/pip ×3 + 中指 ×3 = **10 电机**，腕锁死；改 config 的 joint/motor 映射 + MJCF 删 ring/pinky 即可
- 本地资产路径：repos\orca_core（控制栈+文档）、repos\orcahand_description\v2\models\mjcf\（MJCF，W1 用 MuJoCo viewer 打开这个）、repos\orca_sim、repos\orca_retargeter、repos\orca_teleop
